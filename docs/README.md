## Visão Rápida

Catálogo digital de produtos de confeitaria artesanal desenvolvido com **Django 5.x** e **TailwindCSS**. A interface é pública (sem autenticação), e o gerenciamento de produtos é feito via **Django Admin**.

Dois modelos: `Category` e `Product`. Três páginas públicas: home, catálogo e detalhe do produto.
